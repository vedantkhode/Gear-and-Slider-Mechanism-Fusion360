# Gear and Slider Kinematic Mechanism - Fusion 360

## Overview
A rotary-to-linear motion conversion mechanism designed using Autodesk Fusion 360.

The mechanism uses:
- Spur gear interaction
- Pin-slot linkage
- Motion-linked joints
- Rotary to translational motion conversion

## Mechanism Specifications

| Component | Specification |
|---|---|
| Large Gear | 24 Teeth |
| Small Gear | 12 Teeth |
| Gear Ratio | 2:1 |
| Motion Type | Rotary to Linear |
| Joint Types | Revolute + Pin-Slot |
